What is jQuery

A said on jQuery.com: "jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling..."

Here's another way to put it: jQuery is a very powerful JavaScript library that helps you do awesome things within the browser without having to worry about cross-browser differences. This means that the jQuery code you write will work seamlessly in Chrome, Safari, Internet Explorer, Firefox and Opera.

Using the jQuery API

Now that you know how to select an element, let's go ahead and change the color of our square. How about a nice LightSkyBlue color? (That's the name of the CSS color.) In order to do this we will use jQuery's CSS API. Here's what we'll need to type:

$('#green-box').css('background-color','lightskyblue')

Can you guess what just happened?

  • $('#green-box') We selected an element with an ID of 'green-box'.
  • .css( We used jQuery's CSS API.
  • 'background-color','lightskyblue' We changed the background-color CSS property of this element to a lightskyblue color.

Protip: You can use the "up" arrow in your console to get the last command you typed. This will save you tons of time.

Protip dos: Even though you can copy/ paste the sample code provided, I strongly suggest you actually type things out. This will help you learn wayyy more. I Promise.

Let's now change the border color to a nice steelblue color. Can you guess how to do this?

$('#green-box').css('border-color','steelblue')

We basically used the same command as above, but instead of changing the background-color we changed the border-color.

Actually, how about we remove the background color altogether by setting it to 'transparent'?

$('#green-box').css('background-color','transparent')

Are you getting the hang of it? See, not as hard as you thought :)

Ready for something amazing? Let's add a hadouken inside our transparent box.

We have a hadouken image conveniently saved at http://i.imgur.com/8NArGWZ.gif. To add this image to our box we'll use an HTML image tag to create the image and use jQuery's HTML API to add the image.

Here's how to do this:

$('#green-box').html('<img src="http://i.imgur.com/8NArGWZ.gif" />')

Another way to add the hadouken would be to use jQuery's Append API.

$('#green-box').append('<img src="http://i.imgur.com/8NArGWZ.gif" />')

Adding our animated Ryu

When a user hover's over our image, we want to animate Ryu as if he was ready to fight. Here is the gif we'll need: http://i.imgur.com/TYvGQPJ.gif. To achieve this we'll use jQuery's Hover API, which can take 1 or 2 functions. Functions you say? What is THAT? Well... a function is basically a block of code you want to execute.

Just a second ago I mentioned that jQuery's Hover API can take 1 or 2 functions here's the difference: If you use 1 function, this function will run when a user hovers over the element. That's it. If you use 2 functions, the 1st function will run when the user hovers over the element and the 2nd function will run when the user "un-hovers" (or stops hovering) over the element.

For this example we'll need to use 2 functions. The first function will change Ryu from a stand still position to an animated one. The second function (which will run when the user stops hovering over our image) will change Ryu from an animated position back to a stand still one.

Below is the code you need to run.

Quick note: Since you're using your console, it's easier to type all of this code in one long line. But, if you prefer to use several lines, you can also hit "option" + "enter" on a Mac to get a new line. Just like before, to run your code simply hit "enter".

$('#my-div img').hover( function() {
    this.src = 'http://i.imgur.com/TYvGQPJ.gif'
}, function() {
    this.src = 'http://i.imgur.com/JFMYx3j.png'
})

Let's go over what's happening in the above block of code.

  • "$('#my-div img')" - First, we select the image that is inside #my-div.
  • ".hover(" - Then use jQuery's hover API.
  • "function() { this.src = 'http://i.imgur.com/TYvGQPJ.gif' }" - We then pass our first function. All this is saying is that we'll change the source (or, as said in HTMl, the "src") for this" element. The "this" here refers to the image inside our div, which we selected 2 steps ago. Here's a detailed explanation on using "this" in JavaScript and jQuery (a tricky concept).
  • "," - Is used to separate our functions.
  • "function() { this.src = 'http://i.imgur.com/JFMYx3j.png' }" - Lastly we pass our second function, which does the same as our previous function, but with a different image.

The Hadouken Pose

We now want to change Ryu's pose when someone clicks on him. To do this we'll use jQuery's Mousedown API. Mousedown only takes 1 function and it will allow us to trigger this function whenever the user presses down on their mouse. Here's how to use it:

$('#my-div img').mousedown( function() {this.src = 'http://i.imgur.com/1F4rSOc.png' });

And here's what's happening:

  • "$('#my-div img')" - First, we select the image that is inside #my-div.
  • ".mousedown(" - Then use jQuery's Mousedown API.
  • "function() { this.src = 'http://i.imgur.com/1F4rSOc.png' }" - We then pass our function which will change the src of the image. This is VERY similar to what we did earlier.

How to install jQuery?

The easiest way to install jQuery is to grab the code from Google. Here is the code snippet:

<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>

Note: Notice that there's no "http:" or "https:" before the "//" in the src of the tag. This ensures that both HTTP and HTTPS pages can load this code. If you're coding locally on your computer you'll need to add "http:" to the beginning of the src to make sure jQuery loads. Here's what your script should look like:

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>

You can now create a basic html file and add the jQuery script inside the <head> tag. In the example below we also added a JavaScript file called 'example.js', this is where we would hypothetically write our jQuery code.

<html>
     <head>
         <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
         <script src="example.js"></script>
     </head>
     <body>
     <!-- Your content goes here -->
     </body>
 </html>
